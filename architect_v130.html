Answer Architect — Instruction Manual

Quick start

  1. Open the app (teacher mode). Use the ⚙️ Settings button to configure the lesson.
  2. Add content: paste Source Text and Focus Question (e.g. in AI tab → generate prompt → copy to ChatGPT/Claude → paste JSON back and Load Data), or Load JSON or build manually in Edit.
  3. The app opens on Quote Select when selection is available; students can switch to Writing Desk when ready.
  4. Turn on Active Modules: Quote Select, Writing Desk. Enabling Quote Select automatically enables free select (tap text) for active levels.
  5. In Level Bank, set Quote target (correct) and Weak quotes (distractors) per level. Quote targets are taken only from Level Bank (no separate setting).
  6. Export: Worksheet (Word) or Student App (standalone HTML).

Modules

  • Quote Select 🎯 — Students choose Strong/Not strong, match evidence type, justify. Level Bank controls how many correct and distractor options appear and the Quote Selection Tab Modes per level.
  • Writing Desk ✍️ — Build sentences (slots from Sentence Structure and banks), preview, Commit to Answer. Practice Mode locks level and turns off Auto progress; Practice Force Auto / Fixed Template simplify the builder.

Settings summary

Practice Mode (auto-disables Auto progress), Practice Level, Level Bank (per level: language tier, complexity, subtlety, quote target, weak quotes, tap text, question), Worksheet Match Mode. Quote length and ellipses are set globally in Controls. Quote Select and Writing Desk are always active. A level is active when it has one or more questions selected in the Level Bank.

Sharing the Student App (hosting + QR code)

Host the exported HTML for free: e.g. Netlify Drop (app.netlify.com/drop — drag and drop the file), Neocities (neocities.org), or GitHub Pages. Use the URL where the lesson opens. Create a QR code at qr-monkey.com: choose URL, paste your lesson link, generate and download. Students scan the QR code to open the lesson on their device.

Top tips

  • Quote Select “not enough choices”: set Quote target and Weak quotes in Level Bank and add enough correct items and distractors in Quote Selection Editor. Use Cap counts to targets / Auto-fill correct quotes to targets.
  • Student App opens on Quote Select when selection is available.
  • Fewer steps for weaker students: use Beginner stages; lower quote target and weak quotes in Level Bank.
  • One level only: Practice Mode (Auto progress turns off automatically), set Practice Level, and assign questions only to that level in the Level Bank.
  • Level Bank defines rules per level; a level is active when it has one or more questions selected. AI prompt uses Level Bank to request the right number of correct quotes and distractors per level.
